# 🚀 Privoo - Secure Messenger with AI

[![Flutter](https://img.shields.io/badge/Flutter-3.5%2B-blue)](https://flutter.dev)
[![Firebase](https://img.shields.io/badge/Firebase-Latest-orange)](https://firebase.google.com)
[![License](https://img.shields.io/badge/License-Source%20Available-red)](LICENSE)

**Privoo** هو تطبيق مراسلة آمن وذكي يجمع بين أقوى بروتوكولات التشفير وأحدث تقنيات الذكاء الاصطناعي.

---

## 🔐 الميزات الأمنية

| الميزة | التقنية |
|--------|----------|
| تشفير E2EE | Double Ratchet Protocol |
| تبادل المفاتيح | X3DH Key Exchange |
| التشفير المتماثل | AES-GCM 256-bit / ChaCha20 |
| إخفاء المرسل | Sealed Sender |
| حماية الشبكة | SSL Pinning |
| كشف الاختراق | Root / Jailbreak Detection |
| مقاومة كمومية | Kyber (ML-KEM) + Dilithium (ML-DSA) |
| التحقق من الهوية | Safety Number (مثل Signal) |
